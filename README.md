# CAPSTONE-RESTAURANT-PROJECT
Capstone messy Restaurant project is a complete end-to-end data analysis project built for restaurant business dataset that contains real- world data problems such as missing values, duplicate records, inconsistent formatting, missing date, extra spaces and incomplete entries.
The goal of this project is to stimulate what real data analysts face in industries and help learners practice the full data analysis workflow using tools like;
*Excel
*Power Query
*SQL
Power BI
The dataset includes restaurant information such as 
Customer
Restaurant
Dish Ordered
Rating
Date of Visit
Feedback
Price of Dish Ordered
Quantity

The dataset was cleaned and transformed before performing analysis and creating dashboards.

OBJECTIVES OF THE CAPSTONE PROJECT
1. Data cleaning tasks which include:
   -Removing duplicate
   -Replacing missing values
   -Correcting data types
   -Standardizing text formats
   -Fixing data inconsistences
   -Removing unnecessary spaces

2. Data Transformation tasks includes:
   -Splitting columns
   -Find & select
   -Merge tables
   -Creating calculated field
   -Formatting dates and currencies
   -Creating categories.
   
3. Explorating Data Analysis(EDA)
   Discover patterns, trends and business insight from the cleaned data like:
   -Top-selling food items
   -Least-selling food items
   -Most profitable products
   -Product category with highest sales
   -Product category with lowest sales
   -Average quantity sold per product
   -Revenue by day
   -Revenue by month
   -Peak sales period.

4. SQL Analysis
   SQL query is use to answer business questions like:
   -Total revenue by month
   Top csutomers
   Average customer spending
   Most ordered products.

5. Dashboard Development
   Build interactive dashboards in excel and power BI to visualize performance.
   Dashboard KPIs may include:
   -Total Revenue
   -Total orders
   -Average order value
   Top products
   -Monthly sales trends
   -Customer segments

6. Business Insight Generation
   Provide recommendations that management can use to improve profitability and operations.
   Examples are:
   -Which products should be promoted
   -Which products should be discontinued
   -Which customer segment generates the most revenue?
   -Which location needs improvement?
   -Best sales periods for marketing campaigns.

EXPECTED DELIVERABLES
At the end of the project, the analyst should  produce:
1. cleaned dataset
2. SQL Queries
3. Pivot tables/charts
4. Power BI dashboard
5. Business insight report
6. Recommendations for management.
